# 💜 Purple Rain



## ⚡ Quick start

```bash
git clone https://github.com/ccassini/Purplerain.git
cd Purplerain
npm install
npm run dev
```

Open `http://localhost:3000` and watch the magic happen! 🪄

## 🛠️ Built with

- React + Vite (for speed)
- Real-time WebSocket connection to Monad
- CSS animations (no heavy libraries)
